/smoothie

Slash command sends your problem to all models. The host AI judges responses and gives you one clean answer.

auto-blend

Plans and PRs automatically reviewed by multiple models before you approve. Zero typing required.

Full context mode. Sends project files, git diff, and docs to all reviewers. Shows cost estimate first.
