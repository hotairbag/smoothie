Query multiple AI models in parallel. Get one blended answer. Works with Claude Code, Codex CLI, and Gemini CLI.

click to copy git clone https://github.com/hotairbag/smoothie && cd smoothie && bash install.sh

/smoothie

Slash command sends your problem to all models. Claude judges the responses and gives you one clean answer.

Auto-blend

Every plan automatically reviewed by multiple models before you approve. Zero effort, better plans.

--deep

Full context mode. Sends project files, git diff, and context docs to all reviewers. Shows cost estimate first.

Usage

/smoothie why is auth broken
Manual blend in Claude Code
/smoothie-pr
Multi-model PR review
smoothie models
Pick or manage models
smoothie auto on
Enable auto-blend for plans
